# Заметки с тегами и без

Напишите программу, которая:
1. Принимает ввод строк от пользователя до ввода `"конец"`
2. Если строка начинается с `"тег "` → сохраняет в словарь в формате `ключ: значение`
3. Иначе (кроме `"конец"`) → сохраняет в список
4. Выводит оба хранилища

**Формат тега:** `тег ключ:значение`

**Пример:**
```
Ввод: привет
Ввод: тег задача:сделать
Ввод: пока
Ввод: тег срок:завтра
Ввод: конец

Вывод:
['привет', 'пока']
{'задача': 'сделать', 'срок': 'завтра'}
```

**Подсказки для реализации:**
- Используйте `while True` с `break` при `"конец"`
- Проверяйте начало строки: `строка.startswith("тег ")`
- Для разделения тега: `строка[4:].split(":", 1)` (удаляем "тег " и делим на 2 части по первому двоеточию)
- Добавляйте в словарь: `словарь[ключ] = значение`
