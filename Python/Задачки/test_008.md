
## Условие задачи

Напишите функцию `sum_digits(message)`, которая принимает один аргумент — строку `message` — и возвращает сумму всех цифр, встречающихся в этой строке. Цифрами считаются символы от '0' до '9'. Если цифр нет, функция возвращает 0.

### Входные данные
- Строка `message` может быть любой длины, включая пустую строку.
- Строка может содержать любые символы: буквы (латиница и кириллица), цифры, пробелы, знаки препинания, специальные символы.
- Цифры могут располагаться в любом порядке, они могут образовывать многозначные числа, но учитывается каждая цифра по отдельности (например, в строке "42" сумма будет 4 + 2 = 6, а не 42).

### Выходные данные
- Целое число (`int`) — сумма всех найденных цифр.

### Примеры

```python
>>> sum_digits("Hello 123")
6          # 1 + 2 + 3 = 6
>>> sum_digits("abc")
0          # цифр нет
>>> sum_digits("5 plus 10 is 15")
12         # 5 + 1 + 0 + 1 + 5 = 12
>>> sum_digits("2026 год")
10         # 2 + 0 + 2 + 6 = 10
>>> sum_digits("")          # пустая строка
0
>>> sum_digits("!!! 42 !!!")
6          # 4 + 2 = 6
```

## мини‑подсказки

1. **Как перебрать все символы строки**  
   Строка в Python — это последовательность символов. Вы можете использовать цикл `for`:
   ```python
   for char in message:
       # здесь работаем с каждым символом
   ```

2. **Как определить, является ли символ цифрой**  
   У строк есть встроенный метод `str.isdigit()`. Он возвращает `True`, если символ — цифра, и `False` в противном случае.  
   Например: `"5".isdigit()` → `True`, `"a".isdigit()` → `False`.

3. **Возврат результата**  
   После завершения цикла верните накопленную сумму.
   ```python
   return res
   ```
