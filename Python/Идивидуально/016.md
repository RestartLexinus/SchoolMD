## Условие задачи

Напишите функцию `add_noise(number)`, которая принимает одно число (целое или вещественное) и возвращает новое число, полученное путём добавления к исходному случайной величины (шума). Величина шума выбирается случайно в диапазоне от **-10%** до **+10%** от исходного числа.

### Требования
- Функция должна использовать модуль `random` (импортируйте его в начале кода).
- Для генерации случайного числа в заданном интервале используйте `random.uniform(a, b)`, где `a` и `b` вычисляются как `-0.1 * number` и `0.1 * number` соответственно.
- Функция возвращает получившееся значение (тип `float`).
- Исходное число может быть любым вещественным, включая отрицательные. Проценты считаются от самого числа (т.е. шум пропорционален числу с учётом его знака).

### Примеры работы
```python
>>> import random
>>> random.seed(42)  # для воспроизводимости примера
>>> add_noise(100)
104.572  # примерное значение, зависит от random
>>> add_noise(-50)
-52.418  # шум может быть как положительным, так и отрицательным
>>> add_noise(0)
0.0      # от нуля проценты — ноль, поэтому результат всегда 0
```

### Примечания
- При `number = 0` диапазон шума вырождается в точку 0, поэтому результат всегда 0.
- Не забудьте импортировать `random` в начале программы.
- Функция должна корректно работать для любых чисел (целых, дробных, положительных, отрицательных).
## Короткие подсказки

1. Импортируйте модуль `random`.
2. Определите функцию `add_noise(number)`.
3. Сгенерируйте случайное число (шум) в диапазоне от `-0.1 * number` до `0.1 * number` с помощью `random.uniform()`.
4. Верните сумму исходного числа и полученного шума.
5. Учтите, что при `number == 0` шум будет равен нулю.
