
### **Задача 1: "Четные числа"**

**Условие:**  
Пользователь вводит целое положительное число. Программа должна вывести все четные числа от 2 до этого числа включительно.

**Пример работы программы:**
```
Введите число: 10
Четные числа от 2 до 10:
2 4 6 8 10
```

```
Введите число: 7
Четные числа от 2 до 7:
2 4 6
```

**Подсказки:**
- Используйте цикл `for` с функцией `range()`
- Для проверки четности применяйте оператор `%` (остаток от деления)
- Можно использовать `range()` с шагом 2 для более эффективного решения
- Не забудьте преобразовать ввод пользователя в целое число

**Что нужно учесть:**
- Что делать, если пользователь ввел число меньше 2?
- Как программа обрабатывает нечисловой ввод?
- Включать ли само введенное число, если оно четное?

**Дополнительные вариации:**
- Выводить числа в столбик вместо строки
- Подсчитать и вывести количество четных чисел
- Выводить четные числа в обратном порядке
- Предложить ввести новый диапазон

**Базовый код для начала:**
```python
n = int(input("Введите число: "))

print(f"Четные числа от 2 до {n}:")
# Здесь должен быть ваш цикл
```

### **Задача 2: "Числа, кратные 7"**

**Условие:**  
Пользователь вводит целое число. Программа должна вывести все числа, кратные 7, которые меньше введенного числа.

**Пример работы программы:**
```
Введите число: 30
Числа, кратные 7, меньше 30:
7 14 21 28
```

```
Введите число: 15
Числа, кратные 7, меньше 15:
7 14
```

**Подсказки:**
- Используйте цикл `for` с функцией `range()`
- Для проверки кратности применяйте оператор `%` (остаток от деления)
- Можно генерировать числа, начиная с 7 и увеличивая на 7
- Рассмотрите возможность использования отрицательного шага в `range()` для отрицательных чисел

**Что нужно учесть:**
- Как обрабатывать отрицательные числа?
- Что выводить, если нет чисел, кратных 7 (например, для числа 5)?
- Как программа реагирует на ввод нуля?
- Обработка нечислового ввода

**Дополнительные вариации:**
- Выводить сумму всех найденных чисел
- Найти наибольшее число, кратное 7, но меньше введенного
- Работать с числами по модулю
- Предложить выбрать другое число для кратности (не только 7)

**Базовый код для начала:**
```python
n = int(input("Введите число: "))

print(f"Числа, кратные 7, меньше {n}:")
# Здесь должен быть ваш цикл
```
