# Есть ли котик

**Условие задачи:**

Напишите программу на Python, которая:

1. Запрашивает у пользователя ввод произвольного текста
2. Проверяет, содержится ли в этом тексте подстрока `'кот'` (в любом месте текста)
3. Если подстрока найдена, выводит на экран слово `"мяяяууу"`
4. Если подстрока не найдена, выводит сообщение о том, что котов нет

**Подсказка:**
- Для проверки наличия подстроки в тексте необходимо использовать оператор `in`
- Пример использования `in`: `if "искомая_строка" in проверяемая_строка:`

**Примеры работы программы:**

*Пример 1:*
```
Введите текст: У меня дома живет кот
мяяяууу
```

*Пример 2:*
```
Введите текст: Я люблю собак
Котов не обнаружено
```

*Пример 3:*
```
Введите текст: Этот котик очень милый
мяяяууу
```

**Дополнительное пояснение:**
Оператор `in` в Python позволяет проверить, содержится ли одна строка внутри другой. Например:
- `"кот" in "котик"` вернет `True`
- `"кот" in "собака"` вернет `False`
