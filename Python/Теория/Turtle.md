# Библиотека Turtle в Python: оживляем код с помощью черепашки

**Модуль turtle** — это встроенная в Python библиотека для создания простой графики. По сути, это виртуальная «черепашка» с пером, которая двигается по экрану и оставляет след, как настоящий художник. Черепашка понимает несколько простых команд, комбинируя которые можно создавать удивительные рисунки.

Turtle идеально подходит для знакомства с программированием и визуализации алгоритмов — он превращает абстрактный код в наглядную картинку.

## Начало работы

Модуль turtle входит в стандартную библиотеку Python, поэтому устанавливать его отдельно не нужно. Всё, что требуется, — это сам Python с поддержкой Tkinter (библиотека для создания графического интерфейса). На большинстве систем она установлена по умолчанию. Для проверки можно запустить простой скрипт:

```python
import turtle
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.done()
```

Если после запуска появилось окно и черепашка нарисовала уголок из двух линий — всё работает. Функция `turtle.done()` завершает программу и оставляет окно открытым.

## Управление движением

Вот ключевые команды для управления черепашкой:

- `forward(расстояние)` — двигаться вперёд
- `backward(расстояние)` — двигаться назад
- `right(угол)` — повернуть направо
- `left(угол)` — повернуть налево
- `goto(x, y)` — переместиться в точку с координатами
- `setheading(угол)` — установить направление движения

По умолчанию черепашка находится в центре экрана в точке (0, 0), смотрит вправо и сразу рисует при движении. Чтобы перемещаться без рисования, нужно поднять перо — об этом чуть позже.

## Первые фигуры

Простейший квадрат рисуется с помощью цикла:

```python
import turtle
for _ in range(4):
    turtle.forward(100)
    turtle.right(90)
turtle.done()
```

Для треугольника достаточно трёх шагов с поворотами на 120 градусов, а круг рисуется всего одной командой:

```python
turtle.circle(50)
```

Обратите внимание: черепашка начинает рисовать круг из нижней точки окружности и движется против часовой стрелки.

## Настройки пера и цвета

Turtle позволяет не только рисовать, но и делать рисунки яркими и красивыми:

- `color(цвет)` — установить цвет линии
- `pensize(толщина)` — задать толщину линии
- `speed(скорость)` — регулирует скорость анимации (0 — максимальная, 1–10 — от медленной к быстрой)
- `fillcolor(цвет)` — цвет заливки фигур

Чтобы залить фигуру цветом, используйте пару `begin_fill()` и `end_fill()`:

```python
turtle.fillcolor("red")
turtle.begin_fill()
for _ in range(4):
    turtle.forward(100)
    turtle.right(90)
turtle.end_fill()
```

Черепашке можно менять внешний вид через команду `shape(имя)`. Доступные варианты: `"arrow"`, `"turtle"`, `"circle"`, `"square"`, `"triangle"`, `"classic"`.

Если черепашка мешает любоваться рисунком, её можно скрыть командой `hideturtle()`.

## Поднятие и опускание пера

Иногда нужно переместить черепашку в новую точку, не оставляя за ней линии. Для этого используются команды:

- `penup()` — поднять перо (дальнейшие движения не оставляют следов)
- `pendown()` — опустить перо обратно

Например, чтобы нарисовать две отдельные линии:

```python
turtle.forward(100)      # рисуем первую линию
turtle.penup()
turtle.backward(50)      # перемещаемся без следа
turtle.pendown()
turtle.forward(100)      # рисуем вторую линию
```

## Интерактивность

Модуль turtle позволяет сделать программу интерактивной — реагировать на нажатия клавиш и клики мыши. Для этого сначала нужно включить прослушивание событий командой `turtle.listen()`, а затем привязать нужные функции к клавишам через `onkeypress()`:

```python
def move_forward():
    turtle.forward(20)

turtle.listen()
turtle.onkeypress(move_forward, "Up")   # стрелка вверх
```

## Несколько черепашек

Для сложных рисунков можно создать несколько черепашек, каждая из которых будет рисовать независимо. Это открывает широкие возможности для анимации и многопоточного рисования.

```python
import turtle
t1 = turtle.Turtle()
t2 = turtle.Turtle()
t1.color("red")
t2.color("blue")
t1.forward(100)
t2.left(90)
t2.forward(100)
```

## Таблица основных команд Turtle

| Команда | Описание |
|---------|----------|
| `turtle.forward(d)` | Пройти вперёд на `d` пикселей |
| `turtle.backward(d)` | Пройти назад на `d` пикселей |
| `turtle.right(angle)` | Повернуть направо на `angle` градусов |
| `turtle.left(angle)` | Повернуть налево на `angle` градусов |
| `turtle.goto(x, y)` | Переместиться в точку с координатами `(x, y)` |
| `turtle.setheading(angle)` | Установить направление (0 — восток, 90 — север и т.д.) |
| `turtle.circle(r)` | Нарисовать круг радиусом `r` |
| `turtle.penup()` | Поднять перо (не рисовать при движении) |
| `turtle.pendown()` | Опустить перо (рисовать при движении) |
| `turtle.color(c)` | Установить цвет линии (`"red"`, `"blue"`, `"#ff0000"` и т.д.) |
| `turtle.pensize(w)` | Установить толщину линии в пикселях |
| `turtle.speed(s)` | Установить скорость (0 — макс., 1–10 — от медленной к быстрой) |
| `turtle.begin_fill()` | Начать запись контура для заливки |
| `turtle.end_fill()` | Закончить запись и залить контур |
| `turtle.fillcolor(c)` | Установить цвет заливки |
| `turtle.shape(name)` | Изменить форму черепашки (`"turtle"`, `"arrow"`, `"circle"` и др.) |
| `turtle.hideturtle()` | Скрыть черепашку |
| `turtle.showturtle()` | Показать черепашку |
| `turtle.write(text)` | Написать текст в текущей позиции |
| `turtle.clear()` | Очистить всё, но оставить черепашку на месте |
| `turtle.reset()` | Очистить всё и вернуть черепашку в начальное состояние |
| `turtle.done()` | Завершить рисование и оставить окно открытым |
| `turtle.bye()` | Закрыть графическое окно |
