### 4. print() –∏ input()

**print()** ‚Äî –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –Ω–∞ —ç–∫—Ä–∞–Ω–µ
**input()** ‚Äî –ø–æ–ª—É—á–∞–µ—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ—Ç –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è

```python
# print() - –ø—Ä–æ—Å—Ç–æ –≤—ã–≤–æ–¥–∏—Ç —Ç–µ–∫—Å—Ç
```

### 5. –¢–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö int –∏ float

**int** (integer ‚Äî —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ) ‚Äî —á–∏—Å–ª–∞ –±–µ–∑ –¥—Ä–æ–±–Ω–æ–π —á–∞—Å—Ç–∏: `5`, `-10`, `1000`

**float** (floating point ‚Äî –ø–ª–∞–≤–∞—é—â–∞—è —Ç–æ—á–∫–∞) ‚Äî —á–∏—Å–ª–∞ —Å –¥—Ä–æ–±–Ω–æ–π —á–∞—Å—Ç—å—é: `3.14`, `-2.5`, `0.0`

```python
# –¶–µ–ª—ã–µ —á–∏—Å–ª–∞ (int)

score = -100  # –î–∞, –±—ã–≤–∞—é—Ç –∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ —Å—á–µ—Ç–∞!

# –î—Ä–æ–±–Ω—ã–µ —á–∏—Å–ª–∞ (float)

# –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –º–µ–∂–¥—É —Ç–∏–ø–∞–º–∏

real_number = int(text_number)  # –ü—Ä–µ–≤—Ä–∞—â–∞–µ–º –≤ —á–∏—Å–ª–æ
print(real_number * 2)   # –¢–µ–ø–µ—Ä—å –º–æ–∂–Ω–æ —É–º–Ω–æ–∂–∏—Ç—å: 200
```

### 6. –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ int –∏ float

–° —á–∏—Å–ª–∞–º–∏ –º–æ–∂–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å –≤—Å–µ –æ—Å–Ω–æ–≤–Ω—ã–µ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:

```python
a = 10
b = 3

print(a + b)    # –°–ª–æ–∂–µ–Ω–∏–µ: 13
print(a - b)    # –í—ã—á–∏—Ç–∞–Ω–∏–µ: 7  
print(a * b)    # –£–º–Ω–æ–∂–µ–Ω–∏–µ: 30
print(a / b)    # –î–µ–ª–µ–Ω–∏–µ: 3.333...

# –°–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:
print(a ** b)   # –í–æ–∑–≤–µ–¥–µ–Ω–∏–µ –≤ —Å—Ç–µ–ø–µ–Ω—å (10¬≥): 1000
print(a // b)   # –¶–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –¥–µ–ª–µ–Ω–∏–µ: 3
print(a % b)    # –û—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è: 1
```

<a id="–∫—Ä–∞—Å–∏–≤–æ–µ-—Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ-–≤-print"></a>
### 7. –ö—Ä–∞—Å–∏–≤–æ–µ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –≤ print()

–°–∞–º—ã–π —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–π –∏ —É–¥–æ–±–Ω—ã–π —Å–ø–æ—Å–æ–± ‚Äî **f-—Å—Ç—Ä–æ–∫–∏**:

```python

# –°—Ç–∞—Ä—ã–π —Å–ø–æ—Å–æ–± (—Å–ª–æ–∂–Ω—ã–π)

# –ù–æ–≤—ã–π —Å–ø–æ—Å–æ–± - f-—Å—Ç—Ä–æ–∫–∞ (–ø—Ä–æ—Å—Ç–æ–π –∏ –∫—Ä–∞—Å–∏–≤—ã–π!)

# –ú–æ–∂–Ω–æ –¥–∞–∂–µ –≤—ã–ø–æ–ª–Ω—è—Ç—å –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –≤–Ω—É—Ç—Ä–∏!
print(f"–ß–µ—Ä–µ–∑ –≥–æ–¥ —Ç–µ–±–µ –±—É–¥–µ—Ç {age + 1} –ª–µ—Ç!")

# –ò –æ–∫—Ä—É–≥–ª—è—Ç—å —á–∏—Å–ª–∞:

print(f"–¢–≤–æ–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç: {score:.0f}")  # –í—ã–≤–µ–¥–µ—Ç: 96
```
