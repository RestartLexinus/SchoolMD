# Задача «Список покупок»

Вася записывает список покупок. Каждый день он добавляет в список один новый продукт. Помогите Васе вести список: программа должна запрашивать у пользователя названия продуктов и добавлять их в список, пока пользователь не введёт пустую строку. После завершения ввода программа должна вывести итоговый список покупок, где каждый продукт указан с порядковым номером (начиная с 1).

**Пример работы:**

```
Введите продукт (или нажмите Enter для завершения): Молоко
Введите продукт (или нажмите Enter для завершения): Хлеб
Введите продукт (или нажмите Enter для завершения): Яблоки
Введите продукт (или нажмите Enter для завершения): 
Ваш список покупок:
1. Молоко
2. Хлеб
3. Яблоки
```

**Требования:**

- Используйте список для хранения продуктов.
- Цикл продолжается до тех пор, пока пользователь не введёт пустую строку.
- Нумерация в выводе должна начинаться с 1.

**Подсказка:** для вывода с нумерацией используйте цикл `for i, item in enumerate(shopping_list, start=1):` или просто счётчик.
