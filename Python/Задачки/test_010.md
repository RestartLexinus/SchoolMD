# Шифр Цезаря

Шифр Цезаря — один из самых древних и известных методов шифрования. Назван в честь Юлия Цезаря, который использовал его для секретной переписки с полководцами. Несмотря на свою простоту, этот шифр отлично подходит для знакомства с основами криптографии и программирования.

## Как работает шифр Цезаря?

Идея предельно проста: каждая буква открытого текста заменяется на букву, стоящую на определённое фиксированное число позиций правее (или левее) в алфавите. Это число называется **сдвигом** или **ключом**.

Например, для русского алфавита со сдвигом 3:
- А → Г
- Б → Д
- ...
- Я → В (замыкаем круг)

Английский алфавит со сдвигом 3:
- A → D
- B → E
- ...
- Z → C

Таким образом, слово **HELLO** при сдвиге 3 превратится в **KHOOR**.

Если сдвиг отрицательный, то сдвиг происходит влево — это используется для расшифровки.

## Математическая основа

Чтобы формализовать шифр, удобно представить буквы числами. Например, для английского алфавита:
- A = 0, B = 1, ..., Z = 25.

Тогда шифрование буквы с числовым кодом `x` выполняется по формуле:
```
E(x) = (x + shift) mod 26
```
где `mod 26` — остаток от деления на 26, чтобы результат оставался в пределах алфавита.

Расшифровка:
```
D(x) = (x - shift) mod 26
```

## Реализация на Python

Напишем простые функции для шифрования и дешифрования текста на английском языке. Будем учитывать:
- Только буквы A-Z и a-z (регистр сохраняется).
- Небуквенные символы (пробелы, знаки препинания) остаются без изменений.

### Шаг 1. Определим алфавит
Для удобства создадим строку с алфавитом и функцию для преобразования буквы в число.

```python
def caesar_encrypt(text, shift):
    """
    Шифрует текст шифром Цезаря с заданным сдвигом.
    """
    result = []
    for char in text:
        if char.isupper():
            # Для заглавных букв: берём код от 'A'
            shifted = (ord(char) - ord('A') + shift) % 26
            new_char = chr(shifted + ord('A'))
            result.append(new_char)
        elif char.islower():
            # Для строчных букв: берём код от 'a'
            shifted = (ord(char) - ord('a') + shift) % 26
            new_char = chr(shifted + ord('a'))
            result.append(new_char)
        else:
            # Небуквенные символы оставляем как есть
            result.append(char)
    return ''.join(result)

def caesar_decrypt(text, shift):
    """
    Дешифрует текст, зашифрованный шифром Цезаря.
    Дешифровка — это шифрование с отрицательным сдвигом.
    """
    return caesar_encrypt(text, -shift)
```

### Объяснение кода
- `ord(char)` возвращает числовой код символа в Unicode.
- Для заглавных букв коды идут подряд: `ord('A')=65`, `ord('B')=66`, ... Поэтому `ord(char) - ord('A')` даёт позицию буквы от 0 до 25.
- Добавляем сдвиг, берём остаток от деления на 26, затем прибавляем обратно `ord('A')` и преобразуем в символ через `chr()`.
- Для строчных аналогично, используем `ord('a')=97`.
- Функция дешифрования просто вызывает шифрование с отрицательным сдвигом.

### Пример использования

```python
text = "Hello, World!"
shift = 3

encrypted = caesar_encrypt(text, shift)
print(f"Зашифровано: {encrypted}")

decrypted = caesar_decrypt(encrypted, shift)
print(f"Расшифровано: {decrypted}")
```

Вывод:
```
Зашифровано: Khoor, Zruog!
Расшифровано: Hello, World!
```



## Ограничения и уязвимости

Шифр Цезаря крайне нестоек к взлому:
- **Всего 25 возможных ключей** для английского алфавита (сдвиг от 1 до 25). Их легко перебрать вручную.
- **Частотный анализ**: в любом языке буквы встречаются с разной частотой (например, в английском самая частая буква — E). Сравнив частоты символов в зашифрованном тексте с эталонными, можно быстро определить сдвиг.

Тем не менее, шифр Цезаря отлично подходит для образовательных целей и как основа для более сложных шифров (например, шифра Виженера).
