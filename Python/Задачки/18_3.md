# Математические операции в Python

Прежде чем приступить к задачам, вспомним основные математические операции, доступные в Python.

## Арифметические операторы

| Оператор | Название       | Пример          | Результат |
|----------|----------------|-----------------|-----------|
| `+`      | Сложение       | `5 + 3`         | `8`       |
| `-`      | Вычитание      | `10 - 4`        | `6`       |
| `*`      | Умножение      | `7 * 2`         | `14`      |
| `/`      | Деление        | `15 / 4`        | `3.75`    |
| `//`     | Целочисленное деление | `15 // 4` | `3`       |
| `%`      | Остаток от деления | `15 % 4`   | `3`       |
| `**`     | Возведение в степень | `2 ** 3` | `8`       |

## Приоритет операций

Python соблюдает стандартный математический приоритет:
1. Скобки `()`
2. Возведение в степень `**`
3. Умножение, деление, целочисленное деление и остаток (`*`, `/`, `//`, `%`) выполняются слева направо
4. Сложение и вычитание (`+`, `-`) — слева направо

Пример:
```python
result = 5 + 2 * 3 ** 2   # 1) 3**2=9, 2) 2*9=18, 3) 5+18=23
```

## Преобразование типов

Функция `input()` всегда возвращает строку. Чтобы выполнить математические операции, нужно преобразовать ввод в число:
- `int(input())` — для целого числа
- `float(input())` — для дробного числа

## Полезные функции

- `abs(x)` — модуль числа
- `round(x, n)` — округление до n знаков
- `pow(x, y)` — то же, что `x ** y`

# Задача 📏 Площадь и периметр прямоугольника

Прямоугольник — одна из простейших геометрических фигур. Его основные характеристики:
- Площадь = длина × ширина
- Периметр = 2 × (длина + ширина)

**Задание:**  
Напишите программу, которая запрашивает у пользователя длину и ширину прямоугольника и выводит его площадь и периметр.

**Входные данные:**  
Два числа (каждое на отдельной строке или в одной строке через пробел — на ваше усмотрение). Можно считать, что вводятся положительные числа.

**Выходные данные:**  
Две строки:
- Сначала площадь 
- Затем периметр 

**Пример работы:**

```
Введите длину: 5.5
Введите ширину: 3.2
Площадь: 17.6
Периметр: 17.4
```

**Подсказки:**
- Для ввода двух чисел можно использовать `float(input())` два раза.
- Не забудьте про умножение и сложение.
