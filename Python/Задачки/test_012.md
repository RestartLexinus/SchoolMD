
# Задача «Сильнейший против слабейших»

В турнире участвуют несколько бойцов, каждый со своей силой (число). Организаторы хотят узнать, насколько сильнейший боец превосходит самого слабого, но с учётом того, что слабейший может объединиться с тенью (удвоенная сила). Помоги им вычислить этот показатель.

## Задание

Напишите функцию `max_minus_twice_min(numbers)`, которая принимает один аргумент — список чисел `numbers` (силы бойцов) — и возвращает результат: **максимальное число минус удвоенное минимальное**.

## Входные данные

- Список `numbers` содержит целые числа (можно также вещественные, но для простоты будем считать целыми).
- Гарантируется, что список не пустой (есть хотя бы один боец).
- Числа могут быть положительными, отрицательными или нулём.

## Выходные данные

- Число (int или float) — результат вычисления `max(numbers) - 2 * min(numbers)`.

## Примеры работы

```
>>> max_minus_twice_min([1, 2, 3, 4, 5])
3          # сильнейший = 5, слабейший = 1 → 5 - 2*1 = 3

>>> max_minus_twice_min([10, 20, 30])
10         # 30 - 2*10 = 10

>>> max_minus_twice_min([-5, 0, 5])
15         # 5 - 2*(-5) = 5 + 10 = 15

>>> max_minus_twice_min([7])
-7         # 7 - 2*7 = -7 (если боец один, он одновременно и сильнейший, и слабейший)

>>> max_minus_twice_min([0, 0, 0])
0          # 0 - 2*0 = 0
```

## Мини‑подсказки

- **Как найти сильнейшего и слабейшего**  
  В Python есть встроенные функции `max()` и `min()`, которые принимают список и возвращают максимальное и минимальное значения соответственно.  
  Пример: `max([3, 7, 2])` → `7`, `min([3, 7, 2])` → `2`.

- **Вычисление результата**  
  После того как вы нашли максимум и минимум, просто подставьте их в формулу:  
  `result = maximum - 2 * minimum`  
  и верните `result` с помощью `return`.

- **Краевые случаи**  
  Если в списке всего одно число, максимум и минимум совпадают — формула работает без дополнительных проверок.   ```
