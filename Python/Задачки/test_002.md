### **Задача: "Угадай число"**

**Условие:**  
Напишите программу, в которой компьютер загадывает случайное число в диапазоне от 1 до 100. Пользователь должен угадать это число. После каждой попытки программа сообщает, больше или меньше загаданное число введенного пользователем. Игра продолжается до тех пор, пока число не будет угадано.

**Пример работы программы:**
```
Я загадал число от 1 до 100. Попробуй угадать!
Твоя догадка: 50
Меньше!

Твоя догадка: 25
Больше!

Твоя догадка: 37
Больше!

Твоя догадка: 42
Поздравляю! Ты угадал число за 4 попытки!
```

**Подсказки:**
- Для генерации случайного числа используйте модуль `random` и функцию `randint()`
- Используйте цикл `while` для повторения попыток до угадывания
- Применяйте условные операторы `if-elif-else` для сравнения чисел
- Не забудьте вести счетчик попыток

**Что нужно учесть:**
- Как программа обрабатывает нечисловой ввод?
- Что происходит, если пользователь вводит число вне диапазона 1-100?
- Корректно ли считается количество попыток?

**Дополнительные вариации:**
- Добавьте ограничение на количество попыток (например, не более 10)
- Реализуйте уровни сложности с разными диапазонами чисел
- Предложите сыграть еще раз после окончания игры
- Сохраняйте рекорд - минимальное количество попыток

**Базовый код для начала:**
```python
import random

secret_number = random.randint(1, 100)
attempts = 0

print("Я загадал число от 1 до 100. Попробуй угадать!")
```
