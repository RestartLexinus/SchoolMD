**Задача:**  
Ученик написал программу для анализа результатов контрольной работы.  
Есть словарь `grades`, где ключ — фамилия ученика, а значение — его оценка (от 2 до 5).  
Нужно:  
1. Найти средний балл всех учеников.  
2. Вывести фамилии учеников с оценкой выше среднего балла.  

Вот код ученика, но в нём несколько ошибок (синтаксических, логических и связанных с работой словаря). Найдите и исправьте их.

```python
grades = {
    'Иванов': 5,
    'Петров': 4,
    'Сидоров': 3,
    'Кузнецова': 5,
    'Смирнов': 2
}

# 1. Нахождение среднего балла
total = 0
for student in grades:
    total += grades(student)
average = total / len(grades)

print("Средний балл:", average)

# 2. Поиск учеников с оценкой выше среднего
above_average = []
for student, grade in grades.items():
    if grade > average:
        above_average.append(student)

print("Ученики с оценкой выше среднего:", above_average)
```
