**Задача: «Анализ успеваемости»**

У вас есть список оценок учеников за четверть. Каждая оценка — это целое число от 2 до 5. Нужно выполнить следующие действия:

1. Найти средний балл класса (среднее арифметическое всех оценок).
2. Определить, сколько учеников получили оценку ниже среднего балла.
3. Вывести на экран сообщение:  
   - Если количество таких учеников больше половины от общего числа, вывести: «Нужно подтянуть знания».  
   - Иначе вывести: «В целом успеваемость хорошая».

**Список доступных инструментов** (вы можете использовать только их):  
- списки (`list`, `append`, индексация, срезы)  
- циклы (`for`, `while`)  
- условные операторы (`if`, `elif`, `else`)  
- функции (`def`, `return`) — можно создавать свои  
- встроенные функции: `len()`, `sum()`, `print()`  
- арифметические операции (`+`, `-`, `*`, `/`, `//`, `%`)

**Задание:**  
Напишите **порядок действий** (алгоритм) словами или в виде псевдокода, как вы будете использовать перечисленные инструменты для решения задачи. Не нужно писать готовый код — только описание шагов, какие данные создаёте, какие циклы и условия применяете, какие функции (если нужны) и для чего.
