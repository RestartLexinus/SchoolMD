### **Задача 2: "Калькулятор расходов"**

**Условие:**  
Напишите программу для учета ежедневных расходов. Пользователь последовательно вводит суммы своих расходов за каждый день. Программа должна:

- Накопливать общую сумму расходов
- При превышении лимита в 1000 рублей выводить предупреждение
- Завершать работу при вводе числа 0 и выводить итоговую сумму

**Пример работы программы:**
```
Введите ваши расходы за день (0 для завершения):
Сумма расхода: 300
Текущая сумма: 300 руб.

Сумма расхода: 450
Текущая сумма: 750 руб.

Сумма расхода: 400
ВНИМАНИЕ! Превышен бюджет! Текущая сумма: 1150 руб.

Сумма расхода: 0
Учет завершен. Общая сумма расходов: 1150 руб.
```

**Подсказки:**
- Используйте бесконечный цикл `while True:` с условием выхода через `break`
- Для накопления суммы используйте переменную-аккумулятор
- Проверяйте два условия: равенство ввода нулю и превышение лимита бюджета
- Не забудьте преобразовывать ввод в числовой формат

**Что нужно учесть:**
- Как программа обрабатывает отрицательные числа?
- Что происходит, если сразу ввести 0?
- Корректно ли отображаются сообщения при разных сценариях?

**Дополнительные вариации:**
- Добавьте категории расходов (еда, транспорт, развлечения)
- Реализуйте возможность установки персонального лимита бюджета
- Сохраняйте историю расходов в список для последующего анализа
