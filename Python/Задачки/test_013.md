# Задача «Волшебное число»

Пользователь вводит целое число. Программа должна выполнить с ним несколько арифметических операций и вывести результаты, а также волшебным образом повторить фразу «всё замечательно» столько раз, сколько равно введённое число.

## Задание

Напишите программу, которая:

1. Запрашивает у пользователя ввод целого числа.
2. Выводит на экран четыре результата (каждый на новой строке):
   - число, увеличенное на 5;
   - число, уменьшенное на 3;
   - число, умноженное на 12;
   - строку `"всё замечательно"`, повторённую столько раз, сколько равно введённое число.

## Входные данные

- Одно целое число (может быть положительным, отрицательным или нулём).

## Выходные данные

- Четыре строки:
  1. Результат `число + 5`
  2. Результат `число - 3`
  3. Результат `число * 12`
  4. Строка `"всё замечательно"`, повторённая `число` раз.  
     *Примечание:* если число равно 0 или отрицательное, результатом будет пустая строка (так работает умножение строки на число в Python).

## Примеры работы программы

**Пример 1:**
```
Введите число: 3
8
0
36
всё замечательновсё замечательновсё замечательно
```

**Пример 2:**
```
Введите число: 0
5
-3
0

```
(после 0 идёт пустая строка, поэтому в выводе будет видна пустая линия)

**Пример 3:**
```
Введите число: -2
3
-5
-24

```
(пустая строка в конце)

## Подсказки

- Для получения числа от пользователя используйте `int(input())`.
- Для вывода результатов используйте `print()`.
- Умножение строки на число выполняется с помощью оператора `*`: `"всё замечательно" * n`.
- Помните, что при умножении строки на `0` или отрицательное число получится пустая строка, которая при выводе даст просто пустую строку (перенос строки останется).
