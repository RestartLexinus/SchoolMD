# Задание Нарисовать домик в Turtle

**Что нужно сделать:**  
Нарисовать домик: квадратная стена (100×100), треугольная крыша (высота 70), круглое окно (радиус 15) в центре стены.

**Обязательные требования:**
- Разные цвета для стен, крыши, окна.
- Заливка фигур (`begin_fill()` / `end_fill()`).
- В конце: `hideturtle()` и `turtle.done()`.

**Дополнительно:**  
Добавить дверь, трубу, солнце, облака или использовать циклы.

**Краткая подсказка по координатам (если начинать из (0,0)):**
- Стена: (0,0) → (100,0) → (100,100) → (0,100).
- Вершина крыши: (50, 170).
- Центр окна: (50, 50).
