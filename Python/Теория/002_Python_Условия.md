### 1. –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –∏ True False bool

–ü—Ä–µ–¥—Å—Ç–∞–≤—å, —á—Ç–æ —Ç—ã –∑–∞–¥–∞–µ—à—å –∫–æ–º–ø—å—é—Ç–µ—Ä—É –≤–æ–ø—Ä–æ—Å—ã, –Ω–∞ –∫–æ—Ç–æ—Ä—ã–µ –º–æ–∂–Ω–æ –æ—Ç–≤–µ—Ç–∏—Ç—å —Ç–æ–ª—å–∫–æ "–î–ê" –∏–ª–∏ "–ù–ï–¢". –í Python —ç—Ç–∏ –æ—Ç–≤–µ—Ç—ã –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è `True` (–ø—Ä–∞–≤–¥–∞) –∏ `False` (–ª–æ–∂—å). –≠—Ç–æ **–ª–æ–≥–∏—á–µ—Å–∫–∏–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö** ‚Äî `bool`.

```python
# –ü—Ä–æ—Å—Ç—ã–µ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è
print(5 > 3)    # True (5 –±–æ–ª—å—à–µ 3?)
print(2 == 1)   # False (2 —Ä–∞–≤–Ω–æ 1?)
print(10 != 5)  # True (10 –Ω–µ —Ä–∞–≤–Ω–æ 5?)

# –°—Ä–∞–≤–Ω–µ–Ω–∏—è —Å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏
age = 12
print(age >= 10)  # True (–≤–æ–∑—Ä–∞—Å—Ç –±–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–µ–Ω 10?)
print(age < 8)    # False (–≤–æ–∑—Ä–∞—Å—Ç –º–µ–Ω—å—à–µ 8?)
```

**–û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è:**
- `>` –±–æ–ª—å—à–µ
- `<` –º–µ–Ω—å—à–µ  
- `>=` –±–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ
- `<=` –º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ
- `==` —Ä–∞–≤–Ω–æ
- `!=` –Ω–µ —Ä–∞–≤–Ω–æ

```mermaid
flowchart TD

    B -- –£—Å–ª–æ–≤–∏–µ True --> C[–í—ã–ø–æ–ª–Ω—è–µ–º –∫–æ–¥ –≤–Ω—É—Ç—Ä–∏ if]
```


### 3. –õ–æ–≥–∏—á–µ—Å–∫–æ–µ –≤–µ—Ç–≤–ª–µ–Ω–∏–µ elif else

–ö–æ–≥–¥–∞ –Ω—É–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Å–ª–æ–≤–∏–π –ø–æ–¥—Ä—è–¥, –∏—Å–ø–æ–ª—å–∑—É–µ–º `elif` (—Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –æ—Ç "else if") –∏ `else`.


```python
    print("–ù—É–∂–Ω–æ –ø–æ–¥—É—á–∏—Ç—å –º–∞—Ç–µ—Ä–∏–∞–ª.")
```

–ö–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç:

```mermaid
flowchart TD
```

- –ü—É—Ç–∞—Ç—å `=` (–ø—Ä–∏—Å–≤–æ–µ–Ω–∏–µ) –∏ `==` (—Å—Ä–∞–≤–Ω–µ–Ω–∏–µ)
