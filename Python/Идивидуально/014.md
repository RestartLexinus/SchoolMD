## **Задача: «Случайный рецепт»**

### **Описание**
У тебя есть кулинарная книга в виде словаря. Нужно написать программу, которая:
1. Выводит список доступных категорий блюд
2. Просит пользователя выбрать категорию
3. Случайно рекомендует блюдо из выбранной категории

### **Что делать**
1. **Создай словарь** с рецептами:
```python
recipes = {
    "завтрак": ["омлет", "блины", "сырники", "овсянка"],
    "обед": ["суп", "паста", "плов", "котлеты"],
    "ужин": ["салат", "запеканка", "суши", "стейк"]
}
```

2. **Напиши программу**, которая:
   - Показывает все категории из словаря
   - Спрашивает: "Выберите категорию (завтрак/обед/ужин):"
   - Если категория есть в словаре — выводит случайное блюдо из неё
   - Если категории нет — выводит случайное блюдо из любой категории

### **Пример работы**
```
Доступные категории: завтрак, обед, ужин
Выберите категорию (завтрак/обед/ужин): > обед
Сегодня готовим: паста
```

```
Доступные категории: завтрак, обед, ужин  
Выберите категорию (завтрак/обед/ужин): > десерт
Такой категории нет! Попробуйте: сырники
```

### **Требования**
- Используй `random.choice()` для случайного выбора
- Используй условия `if-else` для проверки категории
- Для вывода списка категорий используй `recipes.keys()`
