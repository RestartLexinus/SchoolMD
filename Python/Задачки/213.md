# Задача «Римские и арабские числа»

Напишите две функции на Python для перевода чисел между арабской (десятичной) и римской системами счисления.

## Теоретическая справка

Римские числа записываются с помощью следующих символов:

| Символ | I | V | X | L | C | D | M |
|--------|---|---|---|---|---|---|---|
| Значение | 1 | 5 | 10 | 50 | 100 | 500 | 1000 |

Правила:
- Символы обычно идут от большего к меньшему, их значения складываются.
- **Вычитающий принцип:** если символ меньшего значения стоит перед большим, то он вычитается.  
  Примеры: IV = 4, IX = 9, XL = 40, XC = 90, CD = 400, CM = 900.
- Один символ не может повторяться более трёх раз подряд.
- Вычитаться могут только I, X, C (символы, обозначающие степени 10).

Диапазон чисел: от 1 до 3999 включительно (римская система не имеет нуля).

## Задание

Реализуйте две функции:

1. **`arabic_to_roman(n)`**  
   - Вход: целое число `n` (1 ≤ n ≤ 3999).  
   - Выход: строка — запись числа римскими цифрами (заглавными буквами).

2. **`roman_to_arabic(s)`**  
   - Вход: строка `s` — корректная запись римского числа (заглавные буквы).  
   - Выход: целое число от 1 до 3999.

## Примеры

| Вызов функции | Результат |
|---------------|-----------|
| `arabic_to_roman(49)` | `"XLIX"` |
| `arabic_to_roman(1996)` | `"MCMXCVI"` |
| `arabic_to_roman(3888)` | `"MMMDCCCLXXXVIII"` |
| `roman_to_arabic("XLIX")` | `49` |
| `roman_to_arabic("MCMXCVI")` | `1996` |
| `roman_to_arabic("MMMDCCCLXXXVIII")` | `3888` |

## Требования

- Запрещено использовать сторонние библиотеки.
- Код должен быть читаемым, переменные названы осмысленно.
- Приветствуются комментарии.
