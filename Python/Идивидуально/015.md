## Условие задачи

Напишите функцию `max_two_sum(numbers)`, которая принимает список чисел (целых или вещественных) и возвращает сумму двух наибольших элементов этого списка: максимального и второго по величине (предмаксимального).

### Входные данные
- Список `numbers`, содержащий одно или несколько чисел.
- Гарантируется, что список содержит хотя бы один элемент. Если элементов меньше двух, функция должна вернуть `None`.

### Выходные данные
- Сумма двух наибольших чисел в списке (тип данных — число).
- Если в списке меньше двух элементов, возвращается `None`.

### Примеры

```python
>>> max_two_sum([3, 7, 2, 9, 5])
16      # 9 + 7
>>> max_two_sum([1, 1, 1])
2       # 1 + 1
>>> max_two_sum([-5, -2, -10, -3])
-5      # -2 + -3
>>> max_two_sum([42])
None    # только один элемент
```

### Примечания
- В списке могут быть отрицательные числа и числа с плавающей точкой.
- Если максимальное число встречается несколько раз, второе по величине также может быть равно ему (как в примере `[1, 1, 1]`).
- Запрещено использовать встроенные функции сортировки (`sort`, `sorted`) и функции поиска максимума (`max`), если задание этого не разрешает явно. В данном случае вы можете реализовать алгоритм самостоятельно, но требование не является обязательным.
